The Stem - Network Performance Testing

This installer will set up The Stem on your Mac.

What gets installed:

  • The Stem binary at /usr/local/stem/stem
  • A launchd service for background operation
  • Configuration files in /usr/local/stem/configs/

After installation:

  • Web UI: http://localhost:8080
  • CLI: /usr/local/stem/stem --help
  • TUI: /usr/local/stem/stem tui
